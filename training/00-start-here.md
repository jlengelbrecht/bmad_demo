# Trainee — Start Here

> Placeholder. Full trainee onboarding content lands in Epic 6.

This is the trainee entry point. By the end of the curriculum you'll be able to lead your team in adopting **BMAD** — a framework for spec-driven AI-assisted development — and you'll understand the team rituals and governance controls that keep AI-generated work from shipping unreviewed.

## Shape of the training

| Component | Time | Format |
|---|---|---|
| 6 in-app lessons | ~15 min each | Self-paced reading with embedded artifact tours |
| 3 hands-on labs | 30–90 min each | One solo, one full-team, one async cross-team |
| 1 capstone | 60–90 min | Run the entire BMAD planning cycle end to end |

Total: roughly 3 hours self-paced, or one half-day if run as a team workshop.

## Begin

The lessons render through this same portal — there is no signup, no account, no remote service. The local user is trusted; the gate that protects shared work is `CODEOWNERS`, taught in Lesson 4.
